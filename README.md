# SuperEco
 Responsive supermarket e-commerce platform with authentication, API use and home delivery service.

## Technologies
* React
* Next.js
* Firebase
* Redux
* Tailwind CSS

### First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
